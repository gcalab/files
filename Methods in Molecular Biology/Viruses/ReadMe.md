# Proteome-wide *ab initio* Structural Analysis of Viral Evolution
### This github repository serves as a supplement to Chapter 10 of the book 'Evolutionary Genomics' (Methods Molecular Biology, Vol. 2981; Springer Nature)

Software used:
1. [Local Colab Fold](https://github.com/YoshitakaMo/localcolabfold) (or latest version of AlphaFold)
2. UCSF Chimera
3. AS2TS server used in CASP
4. USAlign
5. IUPred2A
6. Bio3D package in R
